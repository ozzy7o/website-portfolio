Technologies used to build my websites

CSS

CSS is a style sheet language that describes the presentation of web pages. Whereas HTML is what forms the structure of a web page, CSS is what we use to style the HTML with colors, backgrounds, font sizes, layout, and more. CSS is one of the core technologies for designing and building websites.

HTML

HTML is a special computer language that describes the structure of a document. With HTML, we can build web pages that contain text, images, and many other types of content.

JavaScript

JavaScript is a programming language that drives the web: from front-end user interface design, to backend server-side programming, you'll find JavaScript at every stage of a web site and web application.

How I became a Web Developer.

  • 1990's-2014

    My Humble Beginnings

    As long as I can remember I've loved technology. I mean it's always fascinated me how everything worked. From the youngest age, I used to break things apart and try to put them back together again, just to see what was inside and how it all worked. At age ten I built my first computer, even added a cool water cooler for the CPU & neon lights. I first got into coding when I attending college at Umass Boston. Being a college student and around colleges like MIT, I became addicted to building websites,

  • March 2015

    Freelance

    Being a Freelance Web Developer gave me experience that I wouldn't have got from school or books. Solving problems for clients and projects gave me the insight needed to becoming a good web developer.

  • December 2016

    Always keeping up with the times

    Technology is always changing, and so does code. Just about every other day we have a new language or framework, syntax change. This is why Developers always have to say up to date and always learn. For me Team Treehouse, an online resource has become a great tool for me to stay up to date and to learn new languages and things I did not. !

  • April 2017

    Growing

    As I become better at coding I am looking for new opportunities and projects to expand my knowledge. I would love to work on challenging projects and implement new languages into my projects. !
